## Products

<details>
<summary><strong>Onramp AI</strong> · Repository onboarding for open source contributors</summary>

<p>
<strong>Problem</strong><br>
Understanding an unfamiliar repository and finding the right first contribution takes too much time and context.
</p>

<p>
<strong>Product</strong><br>
Onramp maps repository structure, uses semantic retrieval to surface relevant issues and generates contextual guidance around files, dependencies and implementation paths.
</p>

<p>
<strong>Ownership</strong><br>
I designed and built the product end to end across product experience, retrieval architecture, backend services and contributor workflows.
